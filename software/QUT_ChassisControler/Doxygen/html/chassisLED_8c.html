void led_flash ( unsigned char  times)

Makes the LED blink a set number of times.

led_flash() Input: times - The amount of times to make the LED blink Returns: none

Makes the LED blink a set number of times. The delay between the state of the LED (on/off) is 50ms. The period of the blinks is 100ms
